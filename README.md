# Buboy — Official Music Website

This is a project made by Swamp Records that is a website for Gainsville based musican **Buboy**, which includes ticket links, contact information, streaming platforms, media, and more. The website is built with Next.js and deployed with Vercel

## Getting Started

Install dependencies:
npm install

Run the development server:
npm run dev

Then open http://localhost:3000 in your browser

## Tech Stack

-Next.js
-Vercel
-TypeScript
-Tailwind CSS
